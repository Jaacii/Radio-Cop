DOWNLOAD
-Installiere "Git Bash"
-Rechts auf den grünen button "Clone or download" klicken und den klick kopieren.
-Auf dem Desktop rechtsklick drücken, git bash auswählen und in die konsole "git clone link" schreiben (ohne "" )


(Wenn man downloaded statt klont, ist der ordner nicht verknüpft mit git. Kann man manuell machen, aber fand ich immer komplizierter)



UPLOAD
-Ich muss euch erst hinzufügen. -> git account machen
-Sachen in den Ordner, der mit git konfiguriert ist, packen. (Zum konfigurieren den clone kram oben machen. )
-in git bash eingeben: git add -A
-dann: git commit -m "..." 
-dann: git push
